## Amazon KDP Book Metadata Generator
A Streamlit web application that generates high-converting Amazon KDP book titles and descriptions using Google Gemini AI. This tool helps self-publishers, authors, and KDP sellers quickly create SEO-friendly metadata based on their book details.

## Features
- Generate a catchy, keyword-optimized book title.
- Rewrite your book description professionally.
- Structured output formatted for Amazon KDP.
- Simple and clean Streamlit interface.
- Powered by Gemini 2.5 Flash via LangChain.

## Tech Stack
- Streamlit – Web interface.
- LangChain – Prompt management.
- Google Gemini API – AI content generation.
- Python – Backend logic.

## How It Works
- Enter your Gemini API Key
- Fill in book details:
    Category
    Book Size (6×9 or 8.5×11)
    Total Pages
    Keywords
    Old Description
    Click Generate Title & Description

- The app returns:
- A new SEO-friendly title.
- A persuasive, professionally written description.
- The AI rewrites the description in clear paragraphs without HTML or bullet points, making it ready to paste directly into Amazon KDP.

## API Key
- You need a Google Gemini API Key.
- Get it from Google AI Studio
- Paste it into the app before generating content.
